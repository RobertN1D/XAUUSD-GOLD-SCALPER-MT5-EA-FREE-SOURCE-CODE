# GOLD-HFT-SCALPER-MT5-Tick-Velocity-DOM-Prop-EA
A pure algorithmic XAUUSD scalper for MT5. Built for low-latency and prop firms, it reads raw market tape via tick velocity and DOM order flow imbalance. Instead of fixed rules, a 9-strategy ranking engine dynamically scores and executes VWAP reclaims, ORBs, and momentum spikes. Includes tick-level trade management and dynamic spread filters.
